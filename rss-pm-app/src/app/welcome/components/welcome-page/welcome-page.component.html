RS School is free-of-charge and community-based education program conducted by The Rolling Scopes developer community since 2013.

Required before the start: JavaScript, Typescript, Git, GitHub (clone, add, commit, push, pull, merge, rebase, work with Pull Request), npm, webpack, CSS3 / HTML5, Chrome DevTools, Figma, understanding the concept of REST API
